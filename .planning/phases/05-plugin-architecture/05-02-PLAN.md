Create `apps/control-plane/src/routes/plugins.ts` following the route style in `routes/templates.ts`.

Concrete route behavior:

1. `GET /api/plugins`
   - Parse optional query `state`.
   - If `state` is present, validate with `PluginLifecycleStateSchema`.
   - Return `store.listPlugins({ state })` or `store.listPlugins()`.

2. `GET /api/plugins/:pluginId`
   - Parse params with `z.object({ pluginId: z.string().min(1) })`.
   - Return 404 `{ message: "Plugin not found" }` when missing.

3. `GET /api/plugins/:pluginId/status`
   - Return `{ pluginId, state, diagnostics, updatedAt, enabledAt, disabledAt, lastReloadedAt }`.

4. `GET /api/plugins/:pluginId/history`
   - Return `store.getPluginLifecycleHistory(pluginId)`.

5. `GET /api/plugins/extensions/enabled`
   - Return `EnabledPluginExtensionsSchema.parse(await store.listEnabledExtensions())`.

6. `POST /api/plugins/discover`
   - Call `discovery.discover()`.
   - Return `{ discovered: result.discovered.map((item) => item.manifest), failed: result.failed }`.
   - Do not mutate store in this route.

7. `POST /api/plugins/register`
   - Parse request body with `PluginManifestSchema`.
   - Call `store.registerDiscovered({ manifest, source: { kind: "inline" } })`.
   - Return 201 with the plugin record.

8. `POST /api/plugins/reload`
   - Call `discovery.discover()`.
   - For each valid discovered plugin:
     - If `store.getPlugin(manifest.id)` exists, call `store.reloadPlugin(manifest.id, manifest)`.
     - Otherwise call `store.registerDiscovered(discoveredItem)`.
   - Return `{ reloaded: PluginRecord[], failed: PluginDiscoveryFailure[] }`.

9. `POST /api/plugins/:pluginId/enable`
   - Call `store.enablePlugin(pluginId)`.
   - Return enabled record.

10. `POST /api/plugins/:pluginId/disable`
    - Call `store.disablePlugin(pluginId)`.
    - Return disabled record.

Implement route helpers:

- `parseOrReply(schema, input, reply)` matching templates route.
- `handlePluginError(error, reply)`:
  - duplicate id -> 409
  - not found -> 404
  - failed plugin enable attempt -> 400
  - validation errors -> 400
  - otherwise rethrow.

```typescript
import type { PluginManifest, PluginRecord } from "@feudal/contracts";
import type { AgentRegistrationInput } from "../agent-registry/types";

export function pluginManifestToAgentRegistrations(
  manifest: PluginManifest
): AgentRegistrationInput[] {
  return manifest.extensionPoints
    .filter((extension) => extension.type === "acp-worker")
    .map((extension) => ({
      agentId: extension.workerName,
      capabilities: extension.capabilities,
      status: "online" as const,
      metadata: {
        pluginId: manifest.id,
        pluginVersion: manifest.version,
        extensionPoint: extension.type,
        displayName: extension.displayName,
        artifactName: extension.artifactName,
        required: extension.required,
        enabledByDefault: extension.enabledByDefault
      },
      isTemporary: false
    }));
}

export function pluginRecordsToAgentRegistrations(
  records: PluginRecord[]
): AgentRegistrationInput[] {
  return records
    .filter((record) => record.state === "enabled")
    .flatMap((record) => pluginManifestToAgentRegistrations(record.manifest));
}
```
