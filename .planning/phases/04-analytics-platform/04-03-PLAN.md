Implement AlertService in apps/control-plane/src/services/alert-service.ts:

**Class structure:**
```typescript
import type { AlertRule, AlertState, AlertEvent, MetricSnapshot, MetricListener, WebhookPayload } from "@feudal/contracts";
import { readFileSync } from "node:fs";
import type { AnalyticsService } from "./analytics-service";

export class AlertService implements MetricListener {
  #rules: AlertRule[];
  #webhookUrl: string | undefined;
  #analyticsService: AnalyticsService;
  #states: Map<string, AlertState> = new Map();
  #pendingAlerts: AlertEvent[] = [];
  #unsubscribe?: () => void;

  constructor(options: {
    rules: AlertRule[];
    webhookUrl?: string;
    analyticsService: AnalyticsService;
  })
}
```

**Core methods:**

1. **start()**: `this.#unsubscribe = this.#analyticsService.subscribe(this)`. Initialize AlertState for each rule as { status: "ok" }.

2. **stop()**: Call `this.#unsubscribe?.()` to unsubscribe.

3. **onMetricSnapshot(snapshot: MetricSnapshot)** (implements MetricListener):
   - For each enabled rule, call `this.#evaluateRule(rule, snapshot)`.

4. **Private evaluateRule(rule: AlertRule, snapshot: MetricSnapshot)**:
   - Get metric value: `const value = snapshot[rule.metricField]`. If null or undefined (avgApprovalLatencyMs case), return early.
   - Perform comparison: use switch on rule.operator ("gt", "gte", "lt", "lte", "eq").
   - Determine `isTriggered`: value comparison against rule.threshold is true.
   - Get current state: `this.#states.get(rule.id)`.
   - If isTriggered:
     - If state.status === "firing": check suppression — if now - state.triggeredAt < rule.suppressionWindowMs, update state to "suppressed", do NOT create new alert. Otherwise keep firing.
     - If state.status === "suppressed": check if suppression window expired (now - state.lastSuppressedAt >= rule.suppressionWindowMs). If expired, re-fire alert; otherwise skip.
     - If state.status === "ok" or "resolved": create alert event. Set state to { ruleId, status: "firing", triggeredAt: now, currentValue: value, threshold: rule.threshold }. Call dispatchAlert(rule, state, "firing").
   - If NOT isTriggered:
     - If state.status === "firing" or "suppressed": auto-resolve. Create resolved alert event. Set state to { ruleId, status: "resolved", resolvedAt: now }. Call dispatchAlert(rule, state, "resolved"). After dispatch, transition to "ok".
     - If state.status === "resolved": transition to "ok".
     - If state.status === "ok": no action.

5. **Private dispatchAlert(rule: AlertRule, state: AlertState, eventStatus: "firing" | "resolved")**:
   - Create AlertEvent: `{ id: crypto.randomUUID(), ruleId: rule.id, ruleName: rule.name, status: eventStatus, message: buildMessage(rule, state, eventStatus), metricValue: state.currentValue, threshold: state.threshold, timestamp: new Date().toISOString() }`
   - If rule.notificationChannels.includes("in-app"): push to #pendingAlerts.
   - If rule.notificationChannels.includes("webhook") && #webhookUrl: POST webhook using fetch (fire-and-forget with .catch() for errors).

6. **buildMessage(rule, state, status)**: e.g., "Task backlog is 25 (threshold: 20)" or "Task backlog has recovered to 15 (was: 25)".

7. **getAlertStates()**: Return Array.from(this.#states.values()).

8. **getPendingAlerts()**: Return copy of #pendingAlerts, then clear it.

9. **Static loadRules(path = "config/alert-rules.json")**: Read and parse JSON, validate each rule with AlertRuleSchema from @feudal/contracts. Return validated AlertRule[].

**Step A: Create alerts route file.** apps/control-plane/src/routes/alerts.ts:

```typescript
import type { FastifyInstance } from "fastify";
import type { AlertService } from "../services/alert-service";
import type { AlertRule } from "@feudal/contracts";

export function registerAlertRoutes(
  app: FastifyInstance,
  options: { alertService: AlertService; rules: AlertRule[] }
) {
  app.get("/alerts/state", async () => {
    return { states: options.alertService.getAlertStates() };
  });

  app.get("/alerts/pending", async () => {
    return { alerts: options.alertService.getPendingAlerts() };
  });

  app.get("/alerts/rules", async () => {
    return { rules: options.rules };
  });
}
```
