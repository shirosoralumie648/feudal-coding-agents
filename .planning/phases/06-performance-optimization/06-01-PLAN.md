<objective>
Make the default control-plane metrics path observable and locally cached without adding external infrastructure.

Purpose: Close the current gap where the route can calculate aggregates but the default app registers it without a store. This plan introduces a tiny in-process `MetricsService`, wires it through `createControlPlaneApp()`, and proves bounded local route performance with tests.

Output: `/metrics` uses the app-scoped task source, caches repeated aggregate reads for a short TTL, keeps token metrics honestly placeholder-only, and has focused regression tests.
</objective>

Create `apps/control-plane/src/services/metrics-service.ts`.

Concrete implementation:
1. Import `type TaskProjectionRecord` from `../persistence/task-read-model`.
2. Define `export interface MetricsTaskSource { listTasks(): Promise<TaskProjectionRecord[]> }`.
3. Define `export interface SystemMetricsSnapshot` with:
   - `timestamp: string`
   - `tasks: { total: number; byStatus: Record<string, number>; recoveryRequired: number; awaitingApproval: number }`
   - `runs: { total: number; byStatus: Record<string, number>; byAgent: Record<string, number> }`
4. Implement `export class MetricsService` with private fields for `source`, `ttlMs`, `now`, and cached `{ expiresAt: number; snapshot: SystemMetricsSnapshot } | undefined`.
5. Constructor signature must be exactly:
   `constructor(options: { source: MetricsTaskSource; ttlMs?: number; now?: () => number })`
6. Default `ttlMs` to `1000` and default `now` to `Date.now`.
7. Implement `async getMetrics(options?: { refresh?: boolean })`.
8. Implement `async refreshMetrics()`.
9. Move the existing aggregate logic from `routes/metrics.ts` into `MetricsService.refreshMetrics()`.
10. Count `recoveryRequired` using `task.recoveryState !== "healthy"` to preserve current route behavior.
11. Count `awaitingApproval` using `task.status === "awaiting_approval"`.

Create tests in `apps/control-plane/src/services/metrics-service.test.ts` using a fake source object with a call counter and deterministic `now`.
