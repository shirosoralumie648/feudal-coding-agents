  <action>
    Implement JSON-RPC 2.0 message creation and parsing per D-02:

    1. createJsonRpcRequest(id, method, params, from, to):
       - Returns AgentMessage with jsonrpc: "2.0"
       - Validates params is object or array
       - Generates UUID for id if not provided

    2. createJsonRpcResponse(id, result, from, to):
       - Returns AgentMessage with result field
       - Includes original request id

    3. createJsonRpcError(id, code, message, data?, from, to):
       - Standard JSON-RPC error codes: -32700 (parse error), -32600 (invalid request), -32601 (method not found), -32602 (invalid params), -32603 (internal error)
       - Optional data field for additional error info

    4. parseJsonRpcMessage(raw: unknown):
       - Validates JSON-RPC 2.0 structure
       - Returns AgentMessage or throws ValidationError
       - Handles both request and notification (no id) patterns

    5. isJsonRpcRequest(message): Type guard for request vs notification
    6. isJsonRpcResponse(message): Type guard for response
  </action>

  </verify>
